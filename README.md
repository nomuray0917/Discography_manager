# 💿 Discography Manager

音楽アーティストやバンドのリリース情報を一括管理し、告知用テキストやWebサイト用HTMLを自動生成するデスクトップアプリです。

---

## 🗓 制作時期
2025年12月

---

## 🚀 インストール・起動方法

本ツールは Windows 用の実行ファイル（.exe）として配布しています。

1. **ダウンロード**: 
   右側のメニューにある **[Releases]** セクションから、最新の `DiscographyManager.zip` をダウンロードしてください。
2. **解凍**: 
   ダウンロードしたZIPファイルを右クリックし、「すべて展開」を選択します。
3. **実行**: 
   展開されたフォルダ内にある **`DiscographyManager.exe`** をダブルクリックして起動します。

> [!CAUTION]
> **「WindowsによってPCが保護されました」と表示された場合**
> 1. 画面内の **「詳細情報」** をクリックしてください。
> 2. 右下に表示される **「実行」** ボタンを押すと起動します。

---

## ✨ 主な機能

* **プロジェクト管理**: 入力内容を `.json` 形式で保存・読み込み可能。
* **トラックリスト編集**: 曲の追加、削除、並び替え（▲/▼）、インスト設定（Inst）が直感的に行えます。
* **バイリンガル対応**: 「日本語タイトル」と「英語タイトル」を分けて入力できます。
* **自動出力**: 
    * リリース順（1st, 2nd...）の自動生成
    * SNS/告知用のプレーンテキスト出力
    * Webサイト掲載用のHTMLコード自動生成

---

## 🛠 使い方

1.  **保存先フォルダの選択**:
    「参照」ボタンから、ファイルを書き出す場所（デスクトップなど）を指定します。
2.  **基本情報の入力**:
    リリース日、何作目か（数字）、種別（Single/Album等）、タイトルを入力します。
3.  **トラックリストの作成**:
    「+ 曲を追加」で入力欄を増やします。**上段がメインタイトル、下段が英語タイトル**です。
4.  **保存と書き出し**:
    * **プロジェクト保存**: 作業を中断・再開したい時に使います。
    * **テキスト書き出し**: 告知用の `.txt` ファイル（HTMLコード同梱）を生成します。

Discography Manager (Windows版)
音楽アーティスト・バンド向けのリリース情報管理ツールです。 Pythonのインストール不要で、exeファイルを起動するだけで簡単にご利用いただけます。

インストール・起動方法
ダウンロード: 公開されている DiscographyManager.zip をダウンロードします。

解凍: ZIPファイルを右クリックし、「すべて展開」を選択して解凍します。

実行: 展開されたフォルダ内にある DiscographyManager.exe をダブルクリックして起動します。

⚠️ Windowsによる保護画面が出た場合
本ソフトは個人開発の署名なきソフトであるため、起動時に「WindowsによってPCが保護されました」という青い画面が表示されることがあります。 その場合は、以下の手順で実行してください。

画面内の 「詳細情報」 をクリック

右下に表示される 「実行」 ボタンをクリック これで次回以降は通常通り起動できるようになります。

主な機能
情報の保存・読込: 入力した内容を .json で保存し、いつでも再編集できます。

トラックリスト管理: 曲の追加、削除、並び替え、インスト設定が容易です。

テキスト出力: 告知用のプレーンテキストと、Webサイト掲載用のHTMLコードを同時に書き出します。

自動フォーマット: 「1st」「2nd」などの序数や、月ごとの最終日を自動で補完します。

使い方
保存先の設定: 「参照」ボタンから、ファイルを書き出すフォルダを選択します。

リリース情報の入力: 日付、タイトル、種別を入力します。

曲目の入力: 「+ 曲を追加」で入力欄を増やし、タイトルを入力します（下段は英語タイトル用です）。

書き出し:

「プロジェクト保存」：次回の編集用にJSONファイルを保存します。

「テキスト書き出し」：告知用のテキストファイルを生成します。

アンインストール
レジストリなどは使用していません。フォルダごと削除するだけで完全に削除されます。
